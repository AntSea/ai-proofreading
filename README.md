# AI味去除（文章降重）

一个让AI生成的文章读起来像真人写的Claude Skill。

## 解决什么问题

用AI写内容省时间，但AI味太重容易被读者发现。这个Skill帮你把AI稿子改成自然的人写风格，降低AI检测率的同时保持可读性。

## 核心功能

### 10大类AI腔识别
系统识别并改写10种常见AI写作特征：
- 套话连篇、模板句式、书面词汇
- 结构机械、态度中立、细节缺失
- 车轱辘话、正确但无用、过度总结、EMOJI滥用

### 两种工作模式
- **快速模式（3分钟）**：紧急修改，去掉最明显的AI味
- **完整模式（15分钟）**：深度优化，三遍审校

### 多平台适配
针对公众号、小红书、知乎、微博等不同平台给出不同建议。

## 使用方法

1. 下载 SKILL.md
2. 放入 Claude Code 的 skills 目录（macOS: `~/.claude/skills/`）
3. 对 Claude 说"去除AI味"或使用 `/ai-proofreading`

## 触发方式

当你说这些时会自动触发：
- "去除AI味"、"降重"、"太AI了"
- "像机器人写的"、"没人味"
- "AI检测率高"、"自然一点"

## 技术细节

- 输出修改后的完整文章
- 列出主要修改点
- 目标AI检测率控制在30%以下
