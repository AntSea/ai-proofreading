# ai-proofreading

去除文章AI味的 Claude Skill，让文章更自然、更像人写的。

## 功能

- 系统化降低文章AI检测率
- 10大类AI腔识别与改写
- 支持快速模式（3分钟）和完整模式（15分钟）
- 适配公众号、小红书、知乎、微博等平台

## 使用方法

1. 下载 `SKILL.md` 文件
2. 放入 Claude Code 的 skills 目录：
   - macOS: `~/.claude/skills/ai-proofreading/SKILL.md`
3. 在 Claude 中使用 `/ai-proofreading` 或说"去除AI味"

## 触发关键词

- 去除AI味、降重、太AI了
- AI检测率高、审校文章
- 自然一点、像机器人写的

## AI腔识别清单

1. 套话连篇
2. AI句式模板
3. 书面词汇
4. 结构机械
5. 态度中立
6. 细节缺失
7. 车轱辘话（新增）
8. 正确但无用（新增）
9. 过度总结（新增）
10. EMOJI滥用（新增）
